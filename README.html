E-valua - Sistema de Evaluación Online

Descripción

E-valua es un sistema completo de evaluación online diseñado para instituciones educativas. Permite crear, administrar y evaluar pruebas de manera eficiente con funcionalidad offline.

Características Principales

✓ Sistema de autenticación con roles (Administrador, Docente, Estudiante)
✓ Dashboard personalizado con estadísticas por rol
✓ Creador de pruebas con múltiples tipos de preguntas
✓ Sistema de evaluación con cronómetro y anti-trampa
✓ Analizador de resultados con estadísticas detalladas
✓ Funcionalidad offline con sincronización automática
✓ Diseño responsivo para móviles y tablets
✓ Exportación de datos en múltiples formatos

Tipos de Preguntas Soportadas

  • Selección Múltiple: Una sola respuesta correcta
  • Múltiple Respuesta: Varias respuestas correctas
  • Verdadero/Falso: Preguntas de tipo booleano
  • Respuesta Abierta: Texto libre para evaluación manual

Funciones por Rol

Administrador

  • Gestión completa de usuarios
  • Configuración del sistema
  • Estadísticas globales
  • Exportación masiva de datos
  • Gestión de copias de seguridad

Docente

  • Crear y editar pruebas
  • Asignar pruebas a estudiantes
  • Revisar resultados y estadísticas
  • Calificar respuestas abiertas
  • Exportar reportes de clase

Estudiante

  • Ver pruebas asignadas
  • Realizar evaluaciones
  • Consultar resultados históricos
  • Ver estadísticas personales

Medidas Anti-Trampa

  • Detección de cambio de pestaña
  • Modo pantalla completa obligatorio
  • Cronómetro con límite de tiempo
  • Aleatorización de preguntas y opciones
  • Registro de actividad sospechosa
  • Bloqueo automático por intentos fallidos
Nota: La aplicación funciona completamente offline. Los datos se sincronizan automáticamente cuando hay conexión a internet.

Instalación

  1. Descargar todos los archivos del proyecto
  2. Abrir index.html en un navegador web
  3. Usar las credenciales proporcionadas para acceder

Soporte Técnico

Para soporte técnico o preguntas sobre el sistema, contacte al administrador de su institución.
